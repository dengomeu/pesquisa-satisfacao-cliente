# Projeto – Sistema de Pesquisa de Satisfação - TudoWeb 🌐

Um programa interativo via terminal desenvolvido em Python para automatizar a coleta e análise de feedback de clientes da agência TudoWeb.

Este projeto foi desenvolvido com o objetivo de otimizar o processo de pesquisa de opinião. O sistema substitui formulários manuais por um fluxo automatizado que coleta os dados dos entrevistados, valida as entradas em tempo real e gera um relatório estatístico completo ao final da execução.

Para a **fase de testes** o código foi inicialmente configurado para capturar a coleta de 10 entrevistados. Na versão final, alterado para 50 entrevistados.

---

## 🧮 Funcionalidades do programa
1. **Cadastro de feedback:** O sistema troca ideia com o cliente, pedindo nome, idade e a nota para o atendimento de forma organizada.
2. **Validação:** Utilizou-se a estrutura _while_ que atua como um filtro: se a opção for inválida, o sistema pede a correção na hora, garantindo que nenhum dado errado estrague o resultado final.
3. **Cíclico:** O código é programado para rodar o número exato de entrevistas (10 entrevistas na fase de testes e 50 na fase final).

### 🚀 Instruções de uso
1. Certifique-se de ter o Python instalado.
2. Clone este repositório ou faça o download do arquivo `pesquisa-satisfacao.py`.
3. Execute o comando: `pesquisa-satisfacao.py`
4. Siga as instruções na tela para inserir os dados do aparelho.
